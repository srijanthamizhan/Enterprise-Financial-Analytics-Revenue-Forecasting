# Enterprise Financial Analytics & Revenue Forecasting (BigQuery)

## Project Overview

This project demonstrates end-to-end financial data analysis using
Google BigQuery.\
It includes data warehouse design, advanced SQL analytics, KPI
reporting, and revenue forecasting using BigQuery ML.

The dataset simulates 5,000+ enterprise financial transactions across
multiple regions, segments, and product categories.

------------------------------------------------------------------------

## Tools & Technologies

-   Google BigQuery (Standard SQL)
-   BigQuery ML (ARIMA_PLUS Forecasting)
-   Partitioned & Clustered Tables
-   Chart.js (Executive Dashboard Visualization)
-   GitHub Pages (Deployment)

------------------------------------------------------------------------

## Key Features

-   Designed a partitioned and clustered financial transactions table
    for scalable analysis
-   Built revenue, profit, and margin reporting queries
-   Calculated Month-over-Month growth using window functions (LAG)
-   Created executive summary KPI views
-   Developed a 6-month revenue forecast model using BigQuery ML

------------------------------------------------------------------------

## Business Questions Answered

1.  Which regions generate the highest revenue and profit?
2.  How is revenue trending month-over-month?
3.  What is the overall profit margin?
4.  How will revenue perform over the next 6 months?

------------------------------------------------------------------------

## How to Run

1.  Open Google BigQuery
2.  Run the setup script (bigquery_setup.sql)
3.  Run the analysis script (financial_analysis.sql)
4.  Open index.html to view dashboard visualizations
